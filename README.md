# task-9-redux-thunk

Учебное React-приложение с Redux и Redux Thunk для работы со списком задач.

## Технологии

- React 18
- Redux 5
- React Redux 9
- Redux Thunk 3
- React Router DOM 6
- JSON Server

## Установка

```bash
npm install
```

## Запуск

1. Запустить API (JSON Server) на `http://localhost:3001`:

```bash
npm run server
```

2. В другом терминале запустить фронтенд на `http://localhost:3000`:

```bash
npm start
```

## Сборка production

```bash
npm run build
```

## Тесты

```bash
npm test
```

На текущий момент в проекте нет тестовых файлов, поэтому `react-scripts test` сообщает `No tests found`.
